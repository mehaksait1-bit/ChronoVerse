📖 About
ChronoVerse is an immersive storytelling website where users travel through different periods of history by simply scrolling.
The experience combines smooth animations, cinematic transitions, interactive visuals, and responsive design to create a modern web experience inspired by award-winning websites.
This project was built to explore modern frontend development while experimenting with interactive storytelling and animation.

✨ Features
🌋 Immersive timeline experience
🎬 Smooth scroll-based storytelling
✨ Beautiful animations and transitions
🌍 Multiple historical eras
📱 Fully responsive design
⚡ Fast performance with Vite
🎨 Modern and clean UI
